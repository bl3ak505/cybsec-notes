# 🧰 Mobile Security Tools
Unlike mobile devices of the past, today’s mobiles come with advanced computing capability and connectivity (smartphones). One can use them to store data, browse the Internet, record videos, send SMS, play games, capture photos, and many other tasks. Therefore, mobile devices have become the major source for intruders to steal data. Various types of mobile security tools are discussed below.

---
## Source Code Analysis Tools 
- **Syhunt Mobile** [https://www.syhunt.com] <br>
  Syhunt Mobile is a security tool that scans Android and iOS applications to detect vulnerabilities in their source code. It performs over 350 checks for Android apps written in Java and more than 240 checks for iOS apps developed in Swift, Objective-C, or C. The tool covers OWASP Mobile Top 10 risks, helping developers, testers, and publishers identify weaknesses and improve the security of mobile applications before release.

## Reverse Engineering Tools 
- **Apktool** [https://apktool.org] <br>
  Apktool is a reverse engineering tool for Android apps that allows analysts to decompile and rebuild APK files. It can decode app resources close to their original form, making it easier to study or modify them. Apktool also supports tasks like handling framework-dependent apps, rebuilding APKs after changes, and debugging Smali code, which simplifies malware analysis and app security testing.

## App Repackaging Detectors 
Repackaging is a process of extracting the details of an app from legitimate app stores such as Google Play Store and Apple Store and modifying the app by injecting malicious code. Then, the app is redistributed for public use as an authentic application. Repackaging can also be done during reverse engineering of an application. 

-  **Appdome** [https://www.appdome.com] <br>
  Appdome is a mobile security platform that protects Android and iOS apps using runtime application self-protection (RASP). It defends against tampering, reverse engineering, method hooking, and repackaging without requiring changes to the app’s source code. With built-in anti-fraud, anti-malware, and anti-bot features, it automates mobile app security. Its integrity and checksum validation ensures apps are not altered, preventing fake or modified versions from running.

## Mobile Pen Testing Toolkits 
- **ImmuniWeb® MobileSuite** [https://www.immuniweb.com] <br>
  ImmuniWeb® MobileSuite is a security testing platform for iOS and Android apps that combines machine learning with manual penetration testing. It supports static, dynamic, and interactive testing, along with software composition analysis (SCA), to uncover vulnerabilities in both mobile apps and their backends. The toolkit integrates with SDLC and CI/CD pipelines, provides tailored remediation guidance, and ensures zero false positives. It also offers detailed reports with risk scoring and references like CVE, CWE, and CVSSv3 to help teams strengthen mobile application security.
